How Agent Skills install

Most of these are open-source folders you drop into ~/.claude/skills/ (global) or a repo's .claude/skills/ (project). Two common paths:

• Skills CLI: npx skills add <github-repo-url> --skill <name> — installs a single skill.

• Plugin marketplace: /plugin marketplace add <owner/repo> then /plugin install <name> — for skills shipped as Claude Code plugins.

Open each skill's source link for its exact instructions. Always skim a skill's SKILL.md and any scripts before installing — they run with your agent's permissions.
