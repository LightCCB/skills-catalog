How Agent Skills install

Most are open-source folders you drop into ~/.claude/skills/ (global) or a repo's .claude/skills/ (project). Two common paths:

• Skills CLI: npx skills add <github-repo-url> --skill <name>

• Plugin marketplace: /plugin marketplace add <owner/repo> then /plugin install <name>

Open a skill's source for exact steps. Always skim its SKILL.md and any scripts before installing — they run with your agent's permissions.
